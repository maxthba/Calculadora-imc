# Calculadora de IMC com Flutter

Este é um projeto simples desenvolvido para praticar **Flutter** e a linguagem **Dart**.  
O aplicativo calcula o **IMC (Índice de Massa Corporal)** a partir do peso e da altura informados pelo usuário.

## 📱 Funcionalidades

- Inserir peso
- Inserir altura
- Calcular o IMC
- Exibir o resultado do IMC
- Interface simples e intuitiva

## 🚀 Tecnologias utilizadas

- Flutter
- Dart

## 🎯 Objetivo do projeto

Este projeto foi desenvolvido como parte do meu aprendizado em **desenvolvimento mobile com Flutter**, com o objetivo de:

- Praticar a criação de interfaces
- Entender a estrutura de um aplicativo Flutter
- Trabalhar com entradas de usuário
- Aplicar lógica de cálculo no aplicativo

## 📊 Fórmula utilizada

O cálculo do IMC é feito utilizando a seguinte fórmula:

IMC = peso / (altura²)

Onde:
- **peso** está em quilogramas (kg)
- **altura** está em metros (m)
